# osound

Produce sound of specific frequency in Windows Console.


## usage

```batch
> osound <frequency> <time>

:: [] -> optional argument
:: <> -> argument value
```

```batch
:: a 500Hz sound for 1s
> osound 500 1000

:: a 1kHz sound for 250ms
> osound 1000 250

:: silence for 10s
> osound 0 10000
```
